# Kanban_Apps

# Getting Started #
To get started you just need to

* clone this repository
* ```cd KasirApps```

## For Backend
* ```cd server```
* ```npm i json-server```
* ``json-server --watch db.json``

## For Frontend
* ``` cd client ```
* ``` yarn install  ``` or ``` npm install ```
* ``` yarn start ```

Enjoy it !!!
